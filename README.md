# Website Tỏ Tình ❤️

Một website lãng mạn được tạo ra để tỏ tình với người thương.

## 📁 Cấu trúc file

- `index.html` - File chính của website
- `style.css` - File định dạng và hiệu ứng
- `script.js` - File xử lý tương tác
- `README.md` - File hướng dẫn này

## 🚀 Cách sử dụng

1. **Mở website:** Mở file `index.html` trong trình duyệt web
2. **Trải nghiệm:** Đọc lời tỏ tình và nhấn vào nút "Có, Em Đồng Ý!" hoặc "Cần Thời Gian..."
3. **Tùy chỉnh:** Bạn có thể chỉnh sửa nội dung trong các file theo ý muốn

## 🎨 Đặc điểm

- ✨ Hiệu ứng lấp lánh như ngôi sao
- 💓 Tim đập nhịp nhàng
- 📱 Responsive (thích ứng với điện thoại)
- 🎭 Hiệu ứng mờ nền (glassmorphism)
- 🌹 Animation mượt mà
- 💬 Hiệu ứng gõ chữ typing

## 🔧 Tùy chỉnh nội dung

### Thay đổi lời tỏ tình:
Mở file `index.html` và tìm thẻ `<p class="message">`, thay thế nội dung trong đó.

### Thay đổi màu sắc:
Mở file `style.css` và thay đổi màu trong các phần `.title`, `.heart`, `.btn`...

### Thêm hiệu ứng:
Mở file `script.js` để thêm các hiệu ứng tương tác mới.

## 💡 Gợi ý thêm

- Thêm nhạc nền trong file `script.js`
- Thêm ảnh của hai người
- Thêm ngày kỷ niệm
- Thêm trích dẫn tình yêu

Website được tạo với tình yêu thương! ❤️
